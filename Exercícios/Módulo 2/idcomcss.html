Seletores Personalizados

Para personalizar seletores, é possível utilizar o seletor ID para diferenciar uma tag das outras.

ID - Serve para personalizar seletores individualmente.

CLASS - Serve para personalizar mais de um seletor com a mesma configuração. Também é possível colocar mais de um CLASS, por exemplo: CLASS="basico" + CLASS="principal" = CLASS="basico principal".
