Formatação de Textos

Negrito / Destaque

<b> - Nesta frase, temos um termo em NEGRITO usando a tag B

<strong> - Nesta frase, temos um termo em STRONG usando a tag STRONG

Itálico / Ênfase

<i> - Nesta frase, temos um termo em Itálico usando a tag I

<em> - Nesta frase, temos um termo em Ênfase usando a tag EM

Texto Marcado

<mark> - Nesta frase, temos um texto marcado usando a tag MARK

Texto Deletado / Texto Inserido

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag DEL.

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag INS.

Texto sobrescrito

Para inserir coisas do tipo x20+3, utilizamos a tag SUP

Texto subscrito

Para inserir coisas do tipo H2O, utilizamos a tag SUB
