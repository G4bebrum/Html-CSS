Tags, quebras de linha, parágrafos e símbolos

<p> - para criar parágrafos

<hr> - Para criar linhas

<u> - Sublinhar textos

<address> - Para endereçar endereços

<style> - Para utilizar parametros de CSS no HTML (Torna-se individual, há não ser que a TAG Style seja dentro de HEAD)

&#x + código da emojipédia - Para criar emojis Emojipedia

<small> - TAG para colocar textos pequenos

<big> - TAG para colocar textos maiores

<pre> - TAG para considerar os espaços dentro do código.

<main> - Como uma DIV, mas para mostrar que esse é o conteúdo principal.

DIV - Para organizar como pastas (Sem significado)

MAIN - Para organizar como pastas com o conteúdo principal.

SPAN - Para organizar como pastas in-line.

HEADER - Cabeçalho.

NAV - Pastas para colocar links.

SECTION - Tipo de div para sessões, categorias...

ARTICLE - Para artigos.
