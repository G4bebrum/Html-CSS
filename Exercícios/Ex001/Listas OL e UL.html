Trabalhando com Listas

Listas Ordenadas

Podemos usar a TAG OL para criar listas ordenadas. Cada item da lista devem estar envelopadas com a TAG LI

  1. Acordar
  2. Tomar café
  3. Estudar programação
  4. Ler 5 páginas
  5. Tocar violão

Listas Não Ordenadas

Podemos usar a TAG UL para criar listar não ordenadas. Cada item da lista devem estar envelopadas com a TAG LI

  • Acordar
  • Tomar café
  • Estudar programação
  • Ler 5 páginas por dia
  • Tocar violão
