Comandos em CSS

Preparei uma lista sobre os comandos CSS mais utilizados (ou que fui aprendendo).

  • background-color - Para mudar a cor de fundo

  • font-size - Para mudar o tamanho da fonte

  • font-family - Para mudar a tipografia

  • color - Para mudar a cor

  • text-align - Para alinhamento no texto.

  • @charset - Para ter compatibilidade com UTF-8.

  • text-decoration - Para fazer uma decoração no texto. Ex: Underline.

  • * - Configurações globais das CSS.

  • background-image: linear gradient / radial-gradient - Para criar gradientes

  • background-attachment - Para fixar o gradiente

  • font-weight - Gordura da fonte. (Possível apenas na Work-Sans.)

  • Font-Style - Para criar o estilo da fonte (itálico).

  • @font-face - Para adicionar uma fonte externa baixada.

  • @import url - Para fontes externas não-baixadas.

  • border-width: - Para adicionar o tamanho da borda.

  • border-style: - Para adicionar o estilo da borda.

  • border-color: - Para adicionar a cor da borda.

  • Margin - Para adicionar uma margem, principalmente se quiser uma caixa centralizada.

  • text-intent - Para criar um espaçamento no início do parágrafo.

  • list-style-type - O tipo de estilo da lista

  • columns - Para criar colunas

        

        em HTML é id = em CSS é #
